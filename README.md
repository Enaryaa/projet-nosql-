## Démarrage du projet

Pour lancer le projet : 
```
git clone xx
cd projet-nosql
docker-compose up -d
```

Pour arrêter le projet : 
```
docker-compose down
```

## Structure

Nous avons un site web fait à l'aide du framwork Flask
Les technologies utilisées pour les bases de données sont :
- Mongodb
- Postegres-sql
